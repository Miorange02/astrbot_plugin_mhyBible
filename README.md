# 原神圣经

## 简介

当用户发送的消息中包含关键词“原神”时, 返回一个原神相关语录,可自定义触发关键词和语录库

## 功能

*   **关键词触发：** 自动检测消息中是否包含“原神”关键词。
*   **自动回复：** 当检测到关键词时，自动回复指定文本。

## 使用方法

在 AstrBot 机器人发送的任何消息中提到“原神”关键词，机器人就会自动回复。

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。

## 示例

**用户：** 今天玩了原神

**机器人：** 你说的对，但是《原神》是由米哈游自主研发的一款全新开放世界冒险游戏。
